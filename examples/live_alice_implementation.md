# Live Alice Implementation

The `live_alice_implementation` directory previously linked to a private companion
repository (`utils_myAlice`) used for day-to-day development of the AI family. That
content is intentionally **not** part of the public SoulSketch repo, since it
contains personal memory packs and machine-specific config.

If you want to see a sanitized example of a live memory pack layout, see
[`alice_memory_pack/`](alice_memory_pack/) and
[`reference_memory_pack/`](reference_memory_pack/) in this directory.

For the protocol that governs how a private state repo relates to this public
skeleton, see the "Public Protocol + Private State" section in the top-level
[`README.md`](../README.md).
